# 桌面宠物 🐾

一个基于 [Electron](https://www.electronjs.org/) 的 Windows 桌面宠物：透明悬浮在桌面最上层，可拖动、可缩放，会冒泡聊天、定时提醒、还能帮你控制连续使用时长。配套一个熊猫/竹子主题的设置面板，支持明暗切换。

> 当前版本 `0.1.0`，目标平台 Windows（x64，NSIS 安装包）。

## ✨ 功能

- **透明悬浮宠物**：无边框、置顶、默认鼠标穿透，只有指针落在宠物不透明像素上时才可交互，不挡你点桌面。
- **拖动与位置记忆**：拖着宠物到任意位置，关掉再开会回到原处；坐标做了屏内钳制，永远不会跑到屏幕外消失。
- **缩放**：0.5×～2.5× 自由缩放，一键复位。
- **互动文案气泡**：单击宠物会蹦出一句话；可定时自动冒泡，文案可在面板自定义。
- **多素材轮播**：导入多张图片/动图，按设定间隔自动轮换形象。
- **定时提醒**：到点弹提醒，支持自定义提醒名、头像与提示音。
- **防沉迷休息**：连续使用到设定时长后提醒你休息，支持空闲检测。
- **独占全屏检测**：你在全屏看视频 / 演示 / 玩游戏时自动隐藏，不打扰。
- **设置面板**：熊猫 + 竹子绿色主题，支持面板内明暗切换；开机自启、置顶、暂停、提示音等开关一应俱全。
- **单实例**：开机自启或重复双击只会有一个宠物，不会多开。

## 🚀 开发运行

需要 [Node.js](https://nodejs.org/)（建议 18+）。

```bash
npm install      # 安装依赖
npm start        # 以开发模式启动（electron .）
```

## 📦 打包

使用 [electron-builder](https://www.electron.build/) 生成 Windows NSIS 安装包：

```bash
npm run dist
```

产物输出到 `dist/`。安装包为非静默、可自选安装目录，会创建桌面与开始菜单快捷方式。

> 仓库不含代码签名证书。如需对安装包签名，自行准备 `.pfx` 证书并通过环境变量 `CSC_LINK` / `CSC_KEY_PASSWORD` 传入后再执行打包（证书与密钥切勿入库）。

## 🗂️ 项目结构

```
src/
├── main/                 主进程
│   ├── main.js           窗口创建、IPC、托盘菜单、各定时器
│   ├── store.js          本地配置持久化（userData/config.json）
│   └── services/         素材 / 提示音 / 全屏检测
├── preload/              各窗口的受控 API 桥接（contextIsolation）
└── renderer/             渲染进程
    ├── pet/              宠物窗（拖动/手势/动画/气泡）
    ├── panel/            设置面板
    ├── reminder/         提醒弹窗
    ├── rest/             休息提醒
    └── shared/           公共样式/工具
```

配置（位置、缩放、素材、文案、提醒、自启等）持久化在系统 `userData` 目录下的 `config.json`，重启自动恢复。
