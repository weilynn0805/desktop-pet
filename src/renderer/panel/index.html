桌面宠物 · 设置

形象与素材

可添加多个素材；2 个及以上时按下方间隔轮播。无素材则显示默认形象。

轮播间隔

每隔 分钟换一个

支持透明 PNG / GIF / APNG / WebP 与 WebM；无透明通道的视频会带矩形背景。

互动文案

默认文案池，每行一句，悬停/冒泡时随机显示。若某素材设了专属文案，会与这里的默认文案一起出现。留空则恢复内置默认。

关闭后宠物只说素材的专属文案；没有专属文案则不说话。

每隔 秒说一句

被戳台词

每行一句，单击宠物时随机显示。留空则恢复默认。

事项提醒

到设定时间会弹出提醒面板（白底黑字、可堆叠、不点「我知道了」不消失）。

添加提醒

防沉迷提醒 v1.1 开发中

连续使用电脑过久时，全屏温和提醒休息。善意提醒，可随时关闭，不强制锁屏。

连续使用 分钟后提醒
空闲 分钟重置计时
休息 分钟后自动结束

以上为默认值（PRD 决策 E），v1.1 上线后可在此调整 + 上传全屏休息素材。
